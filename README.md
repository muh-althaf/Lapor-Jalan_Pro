# LaporJalan Pro

Prototipe web statis interaktif untuk sistem pelaporan parkir liar, peta parkir resmi, dashboard petugas, eskalasi instansi, indeks kepatuhan kendaraan, dan live progress tracking.

## Cara menjalankan

1. Ekstrak file ZIP.
2. Buka `index.html` langsung di browser modern, atau jalankan dengan Live Server.
3. Semua data demo tersimpan di browser melalui `localStorage`.

## Modul fitur

- Smart Report Camera: kamera browser bila diizinkan, simulasi deteksi plat, unggah bukti, capture demo.
- Impact-Based Reporting: pilihan dampak, prioritas otomatis, mode anonim, GPS.
- Real-Time Parking Slot: peta demo, slot parkir, zonasi warna, halte ojol.
- Dispatcher Dashboard: antrean laporan, verifikasi, update status, eskalasi lintas instansi.
- Live Progress Tracking: timeline 5 tahap.
- Vehicle Compliance Index: rapor kendaraan berdasarkan riwayat pelanggaran.
- QA Agent: pengecekan fitur front-end demo.

## Catatan pengembangan

Fitur AI plat nomor, AR camera, peta real-time, notifikasi petugas, multi-user, dan database instansi membutuhkan backend/API tambahan agar siap produksi.
